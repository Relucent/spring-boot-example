# spring-boot-example

Spring Boot 示例项目 

```
mvn clean package
java -jar target/spring-boot-hello-0.0.1-SNAPSHOT.war
java -Dloader.path="lib/" -jar target/spring-boot-hello-0.0.1-SNAPSHOT.war
```

Spring Boot 2.x
